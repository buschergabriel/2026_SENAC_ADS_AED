## 🎯 Como Usar Este Repositório

1. **Para cada aula**: Acesse a pasta da aula (ex: `aula-01-memoria-tipos/`)
2. **Veja a estrutura**: `No.py`, `main.py`, `exercicios/` (se houver)
3. **Execute o código**: `python3 main.py`
4. **Estude os exercícios**: Resolvidos ou propostos
5. **Adicione suas anotações**: No `notas.txt`

## 🛠️ Pré-requisitos

- **Python 3.8+** (recomendado 3.10+)
- **Terminal/CMD**
- **Conhecimento**: Lógica de programação básica

## 📖 Estrutura de Cada Aula

Cada pasta de aula segue este padrão:

- **README.md**: Resumo visual + tabela de conceitos + perguntas-chave
- **No.py** ou **[classe].py**: Implementação da estrutura de dados
- **main.py**: Exemplos de uso e testes
- **exercicios/**: Exercícios propostos e/ou resolvidos (se houver)
- **notas.txt**: Suas anotações pessoais

## 🤔 Perguntas-Chave Por Aula

### Aula 01: Memória e Tipos em Python
- [ ] Como Python aloca memória para variáveis?
- [ ] O que é um tipo mutável vs imutável?
- [ ] Como usar `id()` para ver o endereço em memória?

### Aula 02: Nós e Listas Encadeadas
- [ ] Como criar uma classe `No` em Python?
- [ ] O que é uma "lista encadeada simples"?
- [ ] Por que usar `None` para o fim da lista?

### Aula 03: Inserção Ordenada
- [ ] Como inserir um nó sem perder referências?
- [ ] Por que inserção no início é O(1)?
- [ ] Como percorrer uma lista encadeada?

### Aula 04: Big O
- [ ] O que significa O(n)?
- [ ] Uma lista encadeada é O(n) ou O(1) para acesso?
- [ ] Qual é melhor: array ou lista?

### Aula 05: Pilhas / Filas
- [ ] Qual é a regra LIFO e FIFO?
- [ ] Uma pilha pode usar uma lista em Python?
- [ ] Como implementar push() e pop()?
